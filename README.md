
# INS System (Intelligent Network Surveillance System)  — Intelligent Network Topology & Anomaly Detection

## Team Members
- Avikrit Chhetri
- Jeswin Mickel

## Problem Statement
Intelligent Network Topology Discovery & Anomaly Detection for IoT

## Project Idea
We are building a system that simulates an IoT network, automatically discovers device topology, and detects anomalies such as traffic spikes, rogue devices, and network failures using machine learning.

## Features (Planned)
- Network simulation (IoT devices, switches, APs)
- Automatic topology discovery
- Real-time anomaly detection using ML
- Live network visualization dashboard
- Alert system for anomalies

## Tech Stack
- Python
- NetworkX
- Scikit-learn
- Streamlit
- Mininet (optional)
